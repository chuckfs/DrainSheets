
## Owner

Permissions:

- Manage users
- Create properties
- Edit properties
- Delete properties
- Upload documents
- View all data
- Manage permissions

---

## Admin

Permissions:

- Create properties
- Edit properties
- Manage prospects
- Manage contacts
- Upload documents
- Add notes

Restrictions:

- Cannot manage owners
- Cannot modify billing

---

## Editor

Permissions:

- View data
- Edit records
- Upload files
- Add notes

Restrictions:

- Cannot manage users
- Cannot delete major records
